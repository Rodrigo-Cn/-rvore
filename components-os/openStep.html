Open Step

CLASSIFICAÇÃO

É importante esclarecer que Open Step não é um sistema operacional. Na verdade, é uma especificação de interface de programação de aplicativos (API) orientada a objetos desenvolvida pela NeXT. Ela fornece uma estrutura para construir interfaces gráficas de usuário (GUIs) e desenvolver aplicativos de software. O OpenStep foi projetado para ser independente de plataforma, permitindo que os desenvolvedores escrevam código que possa ser executado em vários sistemas operacionais, incluindo NeXTSTEP, Windows NT e diversos sistemas baseados em Unix.

DIFERENÇAS

O OpenStep introduz novas classes e capacidades de gerenciamento de memória.
