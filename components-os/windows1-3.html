Windows 1.0 - 3.0

CLASSIFICAÇÃO

O Windows 1.0-3.1 foram os primeiro lançamentos da Microsoft em 1985 no Estados Unidos, eles Continham o Reversi (jogo), um calendário, bloco de notas, calculadora, relógio, command prompt , Write, Control Panel, Paint e programas de comunicação. Permite a utilização de mouse, janelas e ícones. No entanto, eram sistemas que rodavam por cima do MS-DOS não possuindo um controle de usuário e serviam apenas como um interface mal feita no momento. Segundo Tanenbaum (2010), a Microsoft escreve cada sistema operacional novo com um novo projeto do 0, assim o windows 1.0 não tem nada parecido com o Windows 11.

DIFERENÇAS

De acordo Deitel (2005), o Windows 1.0 foi inspirado na interdade GUI, no Windows 2.0 (1987) um problema de sobreposição de janelas foi resolvido e também o de sobrescrição de programas na memória. O lançamento do Windows 3.0 marcou um importante avanço para a Microsoft no início da década de 1990. Rapidamente se tornou um sucesso comercial, vendendo mais de dois milhões de cópias nos primeiros meses após seu lançamento e alcançando a marca impressionante de 10 milhões de cópias vendidas em apenas dois anos, tanto para empresas quanto para usuários domésticos. Este sistema operacional coincidiu com o surgimento dos processadores 386 da Intel, elevando significativamente o desempenho dos computadores da época. Com o processador 386, o Windows demonstrou um desempenho notável, tornando os computadores mais rápidos e eficientes. Na época, os computadores geralmente tinham entre 2 e 4 megabytes de RAM, uma fração mínima em comparação com a capacidade de memória dos dispositivos atuais. O Windows 3.1, uma evolução do 3.0, ficou conhecido por seus jogos inclusos, como Paciência e Campo Minado, que se tornaram populares entre os usuários regulares e são lembrados até hoje (Jonasson, 2016). Já o Windows 3.0 (1990) foi logo trocado pelo 3.1 (1991) que tinha como melhoria o acesso das aplicacações a mais memória da máquina. Por fim o Windows NT trouxe suporte a redes principalmente as redes locais(LANs).
