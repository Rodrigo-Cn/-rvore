Windows Phone

CLASSIFICAÇÃO

Como dizem que a concorrência é a alma do negócio foi assim que a Microsoft tentou concorrer na área mobile tentando replicar o seus sistemas em celulares, no inicio com o surgimento do conceito pockets PCs o Windows Mobile ganhou um nome de destaque. No entanto, o projeto passou a não dá certo com uma grande heterogeneidade de dispositivos, além de processadores diferentes invibializaram esse projeto. A Microsoft ainda tentou uma escalada na área mobile com o Windows Phone 7 (MÔNACO, 2012). A interface do Windows Phone era limpa, ousada e fluida, focando no conteúdo do usuário. Com o Windows Phone 8, a Microsoft expandiu suas capacidades, mantendo a filosofia de reduzir a desordem e facilitar a conclusão de tarefas. Inspirada na sinalização de transporte de massa, a interface utilizava movimentos táticos únicos. O modelo XNA foi descontinuado, mas jogos existentes do Windows Phone 7.x continuarão funcionando no 8.0, com algumas funcionalidades adicionais disponíveis (WHITECHAPEL el at., 2013). Atualmente é evidente que o projeto não deu liga e o que predomina no mercado é o sistema operacional Android e o Ios.
