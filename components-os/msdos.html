MS-DOS

CLASSIFICAÇÃO

Foram encontradas poucas informações sobre o sistema operacional 86-DOS, desse modo faz melhor começar com o foco no MS-DOS. Segundo Duncan (1988), O MS-DOS era um simples carregador de de programas e evoluiu para um sofisticado sistema para computadores domésticos. ele teve como pai o Sistema Operacional 86-DOS que foi escrito por Tim Paterson em 1980. A Microsoft adquiriu o direito do 86-DOS e usou como base para o lançamento do MS-DOS versão 1.0. O sistema não apresenta uma interface amigável com o usuário, sua principal função era o gerenciamento de arquivos. A versão 2.0 do MS-DOS de 1983 veio com a principais mudanças o suporte para disquetes e discos rígidos e o suporte para comandos personalizados do usuário(shells). Por fim, a versão 3 tinha como principais melhorias os relatórios de erros estendidos e o suporte para compartilhamento de arquivos

DIFERENÇAS

Ele evoluiu de 8 bits do 86-DOS para 16 bit no MS-DOS, além de melhorar o sistema de gerenciamento de arquivos e de alocação de memória e também com a criação de um sistema de arquivos para comandos sequenciais (DUNCAN, 1988).
