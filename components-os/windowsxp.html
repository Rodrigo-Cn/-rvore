Windows XP

CLASSIFICAÇÃO

O Windows XP foi lançado em 2001 como uma ligeira atualização do windows 2000, tendo uma vida de 6 anos levando a rápida depreciação do windows 98 e sendo o sistema operacional mais famoso da época. Ele possui uma interface gráfica mais agradavel com o objetivo de atrair mais consumidores e proporcionando a Microsoft o fim da era do windows baseado em MS-DOS (Tanenbaum, 2010).

DIFERENÇAS

O windows XP foi considerado um sistema complexo com milhões de linhas de código, ele fornece um reservatório de thread para cada processo. Ele possuia um firewall de proteção de rede chamado Internet Connection Firewall. Além disso, ele possui uma GUI melhorada e um melhor acesso a multimídia e possuia suporte tanto para 32 bits quanto para 64 bits. (Deitel, 2005). Segundo Ficarra (2002), o Windows XP tazia a necessidade de utilizar uma senha para o administrador fazer edições no sistema e possuia um bug no Explorer versão 6 por ausencia do Java instalado. O Windows XP trouxe uma ênfase renovada na interface do usuário, introduzindo significativas melhorias com novos ícones, menus e opções que ofereciam aos usuários a capacidade de gerenciar uma ampla gama de tarefas de forma mais eficiente. Além das melhorias visuais, o Windows XP também trouxe aprimoramentos significativos em termos de velocidade e desempenho, juntamente com uma variedade de novas funcionalidades, como criptografia de arquivos do sistema, suporte à rede Wi-Fi, capacidade de acesso remoto e uma versão de 64 bits do sistema operacional (Moinhos, 2020).
