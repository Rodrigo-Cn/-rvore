Windows Mobile

CLASSIFICAÇÃO

De acordo com Moreira (2011), o Windows Phone é um “sistema móvel da Microsoft que se tornou mais uma opção de compra para os usuários brasileiros, oferecendo uma nova proposta de uso com uma interface simples e funcionalidades integradas ao próprio Windows”. Em outras palavras, o Windows Phone é um sistema operacional desenvolvido especificamente para celulares, semelhante ao iOS e Android, mas produzido pela Microsoft. Sistemas operacionais Microsoft Windows também são aplicáveis ao Windows Mobile, incluindo o entendimento dos sistemas de arquivos FAT e dos arquivos index.dat. Assim como em computadores desktop ou laptops, dispositivos Windows Mobile mantêm informações substanciais sobre as atividades do usuário, as quais podem ser relevantes em uma investigação digital, como histórico de navegação na Web, arquivos criados pelo usuário e entradas no Registro. Além disso, os nomes dos computadores conectados recentemente e pontos de acesso WiFi também podem ser identificados. (CASEY, 2010).

DIFERENÇAS

De acordo com Pogue (2011), as inovações são página inicial (Home) do sistema apresenta duas colunas de ícones coloridos, cada um representando algo adicionado pelo usuário para facilitar o acesso a determinados recursos, como um aplicativo, um contato para discagem rápida, uma lista de músicas ou um arquivo de e-mails. Essas colunas são formadas por blocos coloridos que se atualizam automaticamente, eliminando a necessidade de abrir o aplicativo específico para obter a informação desejada.
