Windows Server

CLASSIFICAÇÃO

O Windows Server são os Sistemas Operacionais da Microsoft utilizados para rodar servidores, possui diversas versões que são Windows Server 2022, Windows Server 2019, Windows Server 2016, Windows Server 2012, Windows Server 2008, HPC Server 2008, Home Server, Home Server 2011, Small Business Server, Essential Business Server, Windows Server 2003, Windows 2000 Server Edition e Windows NT Server. É importante salientar sobre as principais versões. De acordo Bermúdez (2020), o Windows Server 2003 foi o sucessor do Windows Server 2000 e tinha como base o núcleo do Windows XP, já o Windows 2008 era baseado no Kernel do Windows NT 6.0. Segundo Deitel (2005), O Windows Server 2003 pode ser utilizado em cluster de alta disponibilidade com até 32 nodos que compartilham seu sistema de armazenamento.

DIFERENÇAS

Segundo Thompson (2012), o Windows Server 2000 foi lançado como uma tentativa da Microsoft recuperar um tempo perdido em relaçao a internet então foi criado um versão do Windows para servidores web com alta facilidade utilizando poucos clicks o problema era que ele apresentava muitas vunerabilidades com muitos hackers invadindo esses servidores e espalhando vírus . O Windows Server 2003 em comparação com o 2003 evoluiu no enfileiramento de mensagens, com o gerenciamento do servidor e da administração e também um novo sistema de backup. No Windows Server 2008 ocorreu a adição da habilidade de autocura para resolução de problemas de arquivos e diretórios corrompidos, leitura e armazenamento de 10 a 30 vezes mais rápido utilizando a tecnologia integrada Hyper-V e 3 versões de controle de servidores que são Serviços de arquivo e impressão, Serviços de domínio e Serviços de aplicativos (MORIMOTO, 2008). O Windows Server 2016 teve como novidade o Windows Hello, adição do Windows Defender, vizualização com Hyper-V, Containers, Nano Server e PowerShell 5.0 (Thompson, 2012). No Windows Server 2019 ocorreu a continuação de uma interface do Windows 10, a utilização nova da infraestrutura hiperconvergente foi disponibilizado um novo centro de administração do Windows, adição de proteção avançada contra ameaças, capacidade de reinicialização suave, integração com o Linux, Adaptador de Rede Azure, VPN sempre ativa e i menu de inicar atualizado (KRAUSE, 2019). Por fim, o Windows Server 2016 teve como novidades uma melhor segurança integrada, novo recursos que simplificam a implantação de controladores de domínio e permitem implementá-los mais rápido, serviço de diretório leve do Active Directory, SET está disponível em todas as versões do Windows Server 2022 que incluem pilha Hyper-V e SDN e começou a focar em um sistema operacional isolado ambiente chamado Dockers. (PANEK, 2023). ,
