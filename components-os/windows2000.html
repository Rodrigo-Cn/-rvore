Windows 2000

CLASSIFICAÇÃO

Segundo Tanenbaum (2010), o que seria o Windows NT 5.0 foi renomeado para Windows 2000, as principais inovações foram o plug-and-play (que facilitou a instalação de novas placas PCI, eliminando a necessidade de mexer nos jumpers), os serviços de diretório de rede (para empresas), a melhoria no gerenciamento de energia (para notebooks) e uma interface gráfica melhorada (para todos).

DIFERENÇAS

As principais novas características foram Suporte para FAT16, FAT32 e NTFS, sistema de arquivos distribuído, novo sistema de backup, Sistema de tolerância a falhas, serviços de acesso remoto e Equilíbrio de carga. (BERMÚDEZ LEÓN. 2020)

REFERÊNCIAS

TANENBAUM, Andrew S. Sistemas Operacionais Modernos. Terceira edição. 2010.

BERMÚDEZ LEÓN, Max José. Windows versiones y características. 2020.
