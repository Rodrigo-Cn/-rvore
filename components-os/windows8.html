Windows 8

CLASSIFICAÇÃO

O Windows evoluiu significativamente do Vista ao Windows 8, com a Microsoft aplicando as lições do Windows 7 para criar um produto bem projetado, com bom desempenho e menos inflado. O Windows 8 usou a abordagem modular MinWin para desenvolver um núcleo de sistema operacional pequeno, expansível para diferentes dispositivos, oferecendo uma experiência comum aos usuários. Essa técnica também foi aplicada ao Windows Phone 8, compartilhando muitos binários com as versões desktop e servidor, e suportando a arquitetura ARM e novos processadores Intel. Contudo, o Windows 8 não foi amplamente aclamado, especialmente pela ausência do botão Iniciar e pela interface tipo tablet em desktops. A Microsoft respondeu com a atualização Windows 8.1 em 2013, corrigindo esses problemas, melhorando a integração com a nuvem e introduzindo novos recursos. Assim, o termo "Windows 8" geralmente se refere ao Windows 8.1, que é a versão discutida. (Tanembaum, 2010).

DIFERENÇAS

A tela inicial do Windows 8, mais bonita e simples que a do Windows 7, exibe apenas um papel de parede, relógio e ícone de rede, exigindo um clique para fazer login. A maior novidade é a interface Metro, padrão no Windows 8, com blocos funcionais que se adaptam tanto a computadores quanto a dispositivos touch. No entanto, a ausência do botão Iniciar foi criticada. O Windows 8.1, reintroduzindo funcionalidades e melhorando a integração com a nuvem. O ambiente desktop, embora acessível via Metro, não é idêntico ao do Windows 7. A pesquisa e as teclas de atalho foram ajustadas, e o sistema agora inclui um antivírus integrado e suporte a processadores ARM, além de inicializar mais rapidamente. Segundo a Microsoft, essas melhorias visam tornar o Windows 8 mais rápido e compatível, oferecendo uma experiência unificada e eficiente em diversos dispositivos.
